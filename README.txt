FrequencyChange
===============

FrequencyChange
===============

Allows for changing MSP430 clock frequency.

Functions
=========

void changeFrequencyOneMhz() - changes the clock frequency to 1 MHz

void changeFrequencyEightMhz() - changes the clock frequency to 8 MHz

void changeFrequencyTwelveMhz() - changes the clock frequency to 12 MHz

void changeFrequencySixteenMhz() - changes the clock frequency to 16 MHz
